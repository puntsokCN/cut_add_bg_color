### 使用方法  

- 本程序需要联网执行！
- 将待抠图的图片 与本程序存放在同一文件夹下
- 运行程序，输入IDkey，输入图片文件名，请加上的对应的后缀名（例如 1.jpg）
- 可在该网址申领IDkey： https://www.remove.bg/api  每个账号可免费处理50张抠图
- 程序里的颜色选项： 白色--w   红色--r  黑色--b
